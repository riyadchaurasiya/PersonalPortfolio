TierSync


  • Tech Stack: SQL, Django, HTML, CSS,

  • Github URL: Project Link

    • TierSync is a project that leverages the Inclusion-Exclusion Principle (IEP) to calculate the unique subscribers from a database, addressing the issue of overlapping subscription data. For example, when a person subscribes to different tiers of an OTT platform like Netflix using the same email, the subscriber may be counted multiple times. Using IEP, this project accurately calculates the number of unique subscribers by eliminating the overlap. The subscriber data is stored in a SQL database, where queries are written to apply IEP and then display the results on a Django-powered website. The website, designed with a Netflix-inspired theme (black and red), includes three pages: a registration/login page, a calculation page, and a welcome page that introduces the project. The database updates with new registrations, and users can see the unique subscriber count and how it was calculated.
